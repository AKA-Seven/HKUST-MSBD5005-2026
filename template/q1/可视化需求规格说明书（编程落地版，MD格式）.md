# 可视化1：2D双轴嵌入聚类动态可视化（带密度晕染）

## 1\. 基础信息

- 名称：企业双维度嵌入时序聚类图

- 类型：2D 动态散点图 \+ 聚类密度晕染

- 核心用途：展示公司相似度（贸易关系\+交易特征）与时序演化

## 2\. 坐标系定义（硬编码维度）

|轴|维度名称|数据类型|数值范围|业务含义|
|---|---|---|---|---|
|X 轴|贸易关系嵌入值|连续浮点数|\[\-1, 1\] 或 \[0,1\]（归一化）|公司合作网络/拓扑结构相似度|
|Y 轴|交易特征嵌入值|连续浮点数|\[\-1, 1\] 或 \[0,1\]（归一化）|交易行为/时序模式相似度|

## 3\. 输入数据字段（必填，程序直接读取）

```json

[
  {
    "company_id": "企业唯一标识",
    "time_slice": "时间切片(如2025-01)",
    "emb_relation": "X轴贸易关系嵌入值",
    "emb_feature": "Y轴交易特征嵌入值",
    "cluster_id": "聚类ID(数字，相同为同一类)",
    "cluster_name": "聚类业务名称(如稳定贸易型)"
  }
]
    
```

## 4\. 视觉渲染规则（编程量化规则）

### （1）散点元素

- 每个公司 = 1个圆形散点；

- 相同 \`cluster\_id\` → 统一固定颜色（配色方案可配置）；

- 散点大小：固定值（建议8\-12px），可按交易金额加权。

### （2）聚类晕染效果

- 渲染方式：核密度估计（KDE）生成聚类区域填充；

- 效果：同聚类散点外围形成半透明渐变晕染，中心深、边缘浅；

- 样式：无边界、柔和融合，覆盖所有同类点的密集区域。

### （3）距离规则

- 两个公司 \`emb\_relation\` \+ \`emb\_feature\` 均相似 → 坐标距离近；

- 任一维度不相似 → 坐标距离远。

## 5\. 时间动态逻辑（平滑动画）

- 时间切片切换时，所有散点 \+ 所有晕染区域同步执行平滑移动；

- 动画参数：过渡时长 500ms\~1000ms（可配置），线性插值；

- 企业状态：
        

    - 新增公司：淡入动画；

    - 消失公司：淡出动画；

    - 存续公司：坐标平滑漂移。

## 6\. 交互功能

- hover散点：显示公司ID、聚类名称、时间切片；

- 时间轴滑块：手动切换时间段/自动播放；

- 图例：点击聚类颜色，隐藏/显示对应类别。

# 可视化2：3D时序贸易关系地形图可视化

## 1\. 基础信息

- 名称：企业关系3D时序地形图

- 类型：3D 曲面地形图 \+ 节点拓扑

- 核心用途：展示公司间邻接关系 \+ 交易关系可信度的时序演化

## 2\. 坐标系定义（硬编码维度）

|轴|维度名称|数据类型|数值范围|业务含义|
|---|---|---|---|---|
|X 轴|公司平面拓扑横坐标|连续浮点数|\[0, 100\]（布局值）|公司邻接关系（模拟邻接表拓扑布局）|
|Y 轴|公司平面拓扑纵坐标|连续浮点数|\[0, 100\]（布局值）|公司邻接关系（模拟邻接表拓扑布局）|
|Z 轴|交易关系嵌入映射值|连续浮点数|自定义归一化|贸易关系强度/嵌入值（地形高度）|
|\-|交易关系可信度|浮点数|\[0, 1\]|地形颜色/透明度映射|

## 3\. 输入数据字段（必填，程序直接读取）

```json

[
  {
    "company_id": "主体公司ID",
    "related_company_id": "关联公司ID",
    "time_slice": "时间切片(如2025-01)",
    "pos_x": "平面X坐标",
    "pos_y": "平面Y坐标",
    "rel_embedding": "Z轴高度(交易关系嵌入值)",
    "confidence": "可信度(0~1)",
    "relation_strength": "关系强度"
  }
]
    
```

## 4\. 视觉渲染规则（编程量化规则）

### （1）平面基底

- X/Y 平面：公司邻接拓扑布局（对应邻接表结构，节点固定/自适应排布）；

- 平面上标记公司节点，节点位置对应拓扑关系。

### （2）3D地形图

- 形态：连续曲面地形图（由公司间关系插值生成）；

- Z轴 = 地形高度 → 越高 = 交易关系嵌入值越大；

- 颜色映射：可信度 0\~1 → 低可信度（红色）→ 高可信度（绿色）；

- 透明度：随可信度动态调整（可信度越高越不透明）。

### （3）关系呈现

- 公司间邻接关系 = 平面拓扑连接；

- 交易关系强度 = 地形高度 \+ 颜色可信度。

## 5\. 时间动态逻辑（平滑动画）

- 时间切片切换时，3D地形曲面 \+ 公司节点同步平滑变形；

- 动画参数：过渡时长 800ms\~1200ms，曲面高度/节点位置线性插值；

- 无跳变、无闪烁，实现地形平滑渐变效果。

## 6\. 交互功能

- 3D操控：鼠标旋转、缩放、平移视角；

- hover地形/节点：显示公司对、关系嵌入值、可信度；

- 时间轴：自动播放/手动拖拽切换时间切片；

- 层级开关：显示/隐藏公司节点、地形等高线。
