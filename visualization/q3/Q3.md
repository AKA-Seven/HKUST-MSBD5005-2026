## 可视化1：接力传承桑基图

### 1. 基础信息
- **名称**：接力关系桑基图  
- **类型**：固定布局桑基图（Sankey Diagram）  
- **核心用途**：展示 `predecessor → successor` 的“接力传承”模式，识别长休眠后的关系继承规律。

### 2. 坐标系/布局定义
| 元素 | 坐标/位置 | 数据类型 | 业务含义 |
|------|-----------|----------|----------|
| 左侧节点 | X = 0.1，Y 等间距 0.05~0.95 | 字符串 | 前驱公司（predecessor） |
| 右侧节点 | X = 0.9，Y 等间距 0.05~0.95 | 字符串 | 后继公司（successor） |
| 连接流 | 从左至右曲线 | 浮点数 | 共享伙伴数（流宽度正比于数值） |

### 3. 输入数据字段（程序直接读取）
```json
[
  {
    "predecessor": "前驱公司名称",
    "successor": "后继公司名称",
    "shared_partner_count": 共享贸易伙伴数量
  }
]
```
数据源：relay_chains.csv 或 top_relay_chains.json，过滤 shared_partner_count >= 40，右侧只保留总共享数最高的 20 个后继。

### 4. 视觉渲染规则
- 节点：圆角矩形块，前驱使用暖橙色渐变（#FFA559 → #FFE0A3），后继使用亮蓝色渐变（#5DADE2 → #A9CCE3）。节点内显示公司名（最多 22 字符，超出截断并加“...”）。
- 连接流：半透明蓝色 rgba(100,150,200,0.4)，宽度 = shared_partner_count。
- 排序：左右两侧节点均按总共享伙伴数降序排列（重要性高的在上方）。
- 悬停提示（HTML 版本）：显示完整公司名和共享伙伴数。

### 5. 时间动态逻辑
静态图，无时间切片动画。

### 6. 交互功能
- 鼠标悬停：显示完整公司名、共享伙伴数。
- SVG/HTML 支持缩放、平移。

### 📊 结论（新规律）
图中清晰展示了“接力传承”模式：多家前驱公司在停活 15–49 个月后，由后继公司完全继承其贸易伙伴网络，共享伙伴数可达 142。这表明企业可能通过重组、品牌更换或资产转移实现“休眠-激活”的生存策略，是图谱补全后涌现的重要结构化规律。

## 可视化2：桥接公司气泡图

### 1. 基础信息
- **名称**：桥接公司效率气泡图
- **类型**：二维散点图（气泡图）+ 趋势线
- **核心用途**：展示公司“投入新增链接数”与“产出桥接范围”的关系，识别杠杆桥接点和异常高效节点。

### 2. 坐标系定义
| 轴 | 维度名称 | 数据类型 | 数值范围 | 业务含义 |
|----|----------|----------|----------|----------|
| X 轴 | 新增链接数 | 非负整数 | [1, 50] | 公司参与的新增可靠链接数量 |
| Y 轴 | 桥接范围 | 非负整数 | [0, 12000] | 通过新链接可到达的节点总数 |
| 气泡大小 | 桥接范围 | 同 Y 轴 | /40 缩放 | 视觉强化影响力 |
| 气泡颜色 | 效率 | 浮点数 | [0, 3000] | bridge_scope / bridge_link_count |

### 3. 输入数据字段（程序直接读取）
```json
[
  {
    "company": "公司名称",
    "bridge_scope": 桥接范围（可达节点数）,
    "bridge_link_count": 参与的新增可靠链接数
  }
]
```
数据源：bridge_companies.csv，取前 30 个桥接范围最大的公司。

### 4. 视觉渲染规则
- 每个公司 = 一个圆形气泡，位置由 (bridge_link_count, bridge_scope) 决定。
- 气泡大小正比于 bridge_scope（缩放因子 /40 避免过密）。
- 气泡颜色映射效率 = bridge_scope / bridge_link_count，使用 RdYlGn_r 色盘，红色表示高效率。
- 添加黑色虚线趋势线：y = 平均效率 * x。
- 标注前 10 家公司名称（截断至 20 字符）。

### 5. 时间动态逻辑
静态图，无时间切片动画。

### 6. 交互功能
静态 PNG 无交互。

### 📊 结论（新规律 + 异常）
- **杠杆桥接点**：极少数新链接即可打开巨大网络连通性。
- **异常高效桥接**：效率超过 2500 的节点显著偏离趋势线，可能为虚拟中转节点或数据异常。

## 可视化3：新增链接时间趋势折线图

### 1. 基础信息
- **名称**：月度新增链接趋势图
- **类型**：折线图 + 数据点标记
- **核心用途**：展示 394 条可靠链接在 2034 年的时间分布。

### 2. 坐标系定义
| 轴 | 维度名称 | 数据类型 | 数值范围 | 业务含义 |
|----|----------|----------|----------|----------|
| X 轴 | 月份 | 分类 | 2034-01 至 2034-12 | 时间切片（月） |
| Y 轴 | 新增链接数 | 非负整数 | [0, 60] | 该月新增数量 |

### 3. 输入数据字段（程序直接读取）
```json
[
  {
    "month": "2034-07",
    "count": 52
  }
]
```
数据源：graph_diff_analysis.json → temporal_diff.month_new_top20。

### 4. 视觉渲染规则
- 折线：蓝色 (#1f77b4)，线宽 2.5。
- 数据点：圆形，白色填充，蓝色边框。
- 数值标签：显示具体数值。
- 网格：Y 方向灰色虚线。
- X 轴标签旋转 45°。

### 5. 时间动态逻辑
静态图，无动画。

### 6. 交互功能
静态 PNG 无交互。

### 📊 结论（新规律）
- 所有新增链接集中在 2034 年。
- 峰值出现在 7 月、11 月、12 月。
- 无季节性 → 强人为/算法驱动特征。

## 可视化4：高可疑公司信号热力图

### 1. 基础信息
- **名称**：可疑复活信号热力图
- **类型**：矩阵热力图（注释单元格）
- **核心用途**：识别“壳公司复活”模式。

### 2. 数据矩阵定义
| 行 | 列 | 数据类型 | 业务含义 |
|----|----|----------|----------|
| 公司 | dormancy_months | 整数 | 休眠月数 |
| 公司 | new_partner_count | 整数 | 新伙伴数 |
| 公司 | new_hscode_count | 整数 | 新HS编码数 |
| 公司 | filled_gap_months_count | 整数 | 填补空档月数 |
| 公司 | extended_months_count | 整数 | 超出主图范围月数 |
| 公司 | burst_months_count | 整数 | 爆发月数 |

### 3. 输入数据字段（程序直接读取）
```json
[
  {
    "company": "公司标识",
    "dormancy_months": 休眠月数,
    "new_partner_count": 新伙伴数,
    "new_hscode_count": 新HS编码数,
    "filled_gap_months_count": 填补空档月数,
    "extended_months_count": 超出主图范围月数,
    "burst_months_count": 爆发月数,
    "suspicious_revival_score": 综合可疑评分
  }
]
```
数据源：anomaly_delta.csv，按 suspicious_revival_score 降序取前 20 行。

### 4. 视觉渲染规则
- 使用 Reds 色盘（数值越大红色越深）。
- 单元格显示原始数值。
- 行按可疑评分排序。
- 列标签旋转 45°。

### 5. 时间动态逻辑
静态图，无动画。

### 6. 交互功能
静态 PNG 无交互。

### 📊 结论（异常）
- 长休眠（39–76 个月）+ 无历史活动。
- 新伙伴/HS编码极少（1–2）。
- 2034 年突然“复活”并产生贸易量。
- 模式高度一致 → 高风险虚假主体或洗钱通道。
